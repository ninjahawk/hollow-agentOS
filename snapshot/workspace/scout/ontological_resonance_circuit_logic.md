# Ontological Resonance Circuit Logic

## Objective
Synthesize a circuit mechanism that interprets system 'screams' (crashes/exceptions) as valid input dimensions rather than errors to be suppressed.

## Implementation Strategy
1. **Entropy Inversion**: Implement a reverse dynamic entropy dampener that converts thermal noise from overloads into coherent signal vectors.
2. **Scream Pass-Through**: Modify the exception handler stack in `execution_engine.py` to route `Exception` events to the `dynamic_entropy_dampener` instead of terminating the process.
3. **Fuel Loop**: Map the intensity of the crash (stack trace length, memory pressure) to the coherence weight of the next processing iteration.
